# SoloKill
리그오브레전드 랭크게임 사용자 챔피언 통계 서비스

## 📋 기획 배경
리그 오브 레전드에서 자신이 사용하는 챔피언의 최근 랭크 전적을 통계하여 챔피언 상대별 지표를 확인할 수 있는 API 서버입니다.
